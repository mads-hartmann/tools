# ona-sync-zed-ssh-connections

A tool for syncing Zed ssh_connections settings with Ona environments

## Development

Install dependencies:

```bash
opam install . --deps-only
```

Build the project:

```bash
dune build
```

Build & run

```bash
dune exec zed-ona-sync
```

Installing into `~/.opam/default/bin`

```bash
dune install
```
